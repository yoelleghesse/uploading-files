This script utilizes the Requests library to send a POST request to the specified URL (https://cgi-lib.berkeley.edu/ex/fup.cgi) with a file attached. It reads the contents of the file (myfile.txt) and sends it as part of the request payload.

Install dependencies:

``pip install requests``

Place the file (myfile.txt) you want to upload in the project folder.

Execute the Python script to start the file upload process.

``python file_upload.py``

Ensure that you have appropriate permissions to access the specified URL and upload files. Adjust the URL (https://cgi-lib.berkeley.edu/ex/fup.cgi) and file name (myfile.txt) as needed.
